# Traffic-Density-Prediction
# AI-Based Traffic Density Prediction Using Machine Learning

## Overview
This project predicts traffic congestion levels (Low, Medium, High) using weather and temporal data from the Metro Interstate Traffic Volume dataset.

## Features Used
- Temperature
- Rainfall
- Snowfall
- Cloud Coverage
- Hour of Day
- Day of Week
- Month
- Weather Conditions

## Machine Learning Model
Random Forest Classifier

## Results
- Dataset Size: 48,000+ records
- Accuracy Achieved: 92.8%

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Google Colab

## Project Workflow
1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis

7. Traffic Density Prediction
